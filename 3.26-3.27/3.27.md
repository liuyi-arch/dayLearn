# 双远程，双分支：仓库1 <-> 本地仓库 <-> 仓库2
```md
# 0. 从远程仓库A fetch分支拉取代码到本地...

# 1. 与双远程仓库建立连接并为远程仓库命名别名

git remote add internal https://远程仓库A

git remote add github https://远程仓库B

# 2. 正常分支与孤儿分支

git checkout -b main # 正常分支

git checkout --orphan main # 孤儿分支

# 说明：正常分支会继承提交历史，孤儿分支不会有之前提交历史

# 3. 场景一：从仓库A（别名internal）拉取最新代码到本地

git checkout develop

git pull internal develop

# 4. 场景二：本地develop分支代码拷贝到本地main分支

git checkout main

git checkout develop -- . # 将develop分支最新代码拷贝到当前所在main分支

# 5. 场景三：本地main分支代码push到远程仓库B（别名github）

git add .

git commit -m 'message'

git push github main
```
